# Project Overview
- **Purpose:** Implements the Good Agent platform, providing agent orchestration, messaging, templating, and utilities for GoodKiwi services.
- **Key Modules:** `core` (event routing, templating, models), `utilities` (helpers like retries, text), `extensions`, `tools`, and `resources`.
- **Notable Features:** Async-heavy architecture, templated MDXL rendering, retry/backoff utilities, and integration with MCP and GoodKiwi tooling.

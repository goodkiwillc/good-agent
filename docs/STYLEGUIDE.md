# Good Agent Style Guide

**"Aggressively Adequate"**

This document outlines the visual and tonal standards for the Good Agent project. Our aesthetic is inspired by mid-century bureaucracy, municipal utility providers, and Ron Swanson's "Very Good Building & Development Co."

---

## 1. Brand Voice

- **Tone:** Deadpan, Stoic, Reliable, Concise.
- **What we are:** A library that works.
- **What we are NOT:** "Delightful", "Magic", "Ninja", or "Rockstar".
- **Keywords:** Functional, Standard, Compliant, Durable.

## 2. Visual Identity

### The Logo: "Bureaucrat Bot"
A square, grey robot with a mustache. It represents the unenthusiastic competence of the software.

### Color Palette

| Color Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **Chassis Grey** | `#E0E0E0` | Primary Backgrounds, Robot Body |
| **Charcoal** | `#2D2D2D` | Text, Borders, Robot Eyes |

---

## 3. Typography

We use standard, sans-serif fonts. We do not need fancy serifs.
- **Headings:** `Roboto`, `Helvetica`, or `Arial`.
- **Body:** `Roboto`, `Helvetica`, or `Arial`.
- **Code:** `Roboto Mono`, `Consolas`, or `Courier New`.

---

## 4. UI Components

- **Buttons:** Square or slightly rounded corners (`2px`). No pill shapes.
- **Shadows:** Minimal. Flat design is preferred.
- **Icons:** Simple, heavy-stroke line icons.
